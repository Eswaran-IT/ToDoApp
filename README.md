# TODO App - Full-Stack Application

---

## Description:

The **TODO app** is a full-stack application that allows users to create, manage, and delete tasks. It demonstrates a modern web application built using **Spring Boot** for the backend and **React** for the frontend, incorporating secure authentication, user interactions, and data handling processes.

---

## Technologies Used

### Backend (**Spring Boot**)

- **Spring Boot** - Framework for building the backend API.
- **Spring Data JPA** - For database interaction with MySQL.
- **MySQL** - Database used to store task data.
- **Lombok** - Optional, used for concise code writing.

---

### Frontend (**React**)

- **React** - JavaScript library for building user interfaces.
- **Material-UI** - For UI components and styling.
- **Redux** - State management library for handling global application state.
- **Axios** - For making API calls to the backend.
